
# Aurielle — Luxury Cosmetics (EN/AR, PWA, QR, Static)

## Quickstart
```bash
npm install
npm run dev
```
Opens `http://localhost:8080` with live server.

## Commands
- `npm run dev` — Build once and serve.
- `npm run watch` — Watch Tailwind & rebuild product pages.
- `npm run build` — Build (legacy).
- `npm run build:prod` — Production build (minified) + copy static.
- `npm run deploy:gh` — Build and publish `/dist` to GitHub Pages.

## Owner Workflow
- Add a product:
  ```bash
  node scripts/add-product.js --id velvet-rouge --enName "Velvet Rouge" --arName "فيلفت روج" --image "images/velvet-rouge.webp"
  npm run build
  ```
- Generate QR codes:
  1. Set `data/config.json` → `"baseUrl"` to your live domain.
  2. `node scripts/generate-qr.js`

- Netlify Deploy: Build `npm run build:prod`, Publish `dist`.

## Architecture
- All copy in `data/translations.json` (EN/AR).
- Language managed by `public/src/i18n.js` (dir, placeholders, titles).
- Product pages built from `data/products.json` via `scripts/build.js`.
- QR codes generated by `scripts/generate-qr.js` (level H).

## Accessibility
- Keyboard nav, ARIA labels, strong contrast, focus rings.
